⚙️ Gmail Auto-Rules

Set up Gmail filters so emails are auto-labeled and picked up automatically on every scan — no manual forwarding needed.
Click Create in Gmail → for each rule, complete the 2-click filter creation in Gmail, then scan as usual.

Pre-built rules
Custom rule
How it works: Gmail will auto-label matching emails as dashboard-revenue or dashboard-expense. The dashboard scans these labels on every ⚡ Scan Now and extracts amounts from the email body automatically. If an amount can't be found, it'll appear in a "⚠️ Needs Review" banner for you to fill in.
📧 How to log items by forwarding email
Forward any email to yourself (your Gmail account) and change the subject to include a tag and dollar amount:
[REVENUE] Airbnb payout $3,200 → added to Revenue panel
[EXPENSE] Home Depot supplies $45.00 → added to Expense Tracker

Then click 📧 Process Forwarded Emails (or ⚡ Scan Now) and the app will find the email, extract the amount and description, and add it automatically. Each email is only processed once.
